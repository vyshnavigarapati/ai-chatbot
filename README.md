# AI Chatbot

An AI Conversational Chatbot built using Python, Streamlit, and OpenRouter API.

## Features
- Interactive chatbot interface
- Real-time AI responses
- Simple and user-friendly UI
- Deployed using Streamlit Cloud

## Technologies Used
- Python
- Streamlit
- OpenRouter API
- OpenAI Library

## Project Structure
- app.py
- requirements.txt
- README.md

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py```
